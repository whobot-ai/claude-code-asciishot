╔═══════════════════════════════════════════════════════════════════════╗
║                       RESTful API 设 计 规 范                         ║
╚═══════════════════════════════════════════════════════════════════════╝

  ┌──────────┬────────────────────┬─────────────────────────────────────┐
  │  方法    │  路径              │  说明                               │
  ├──────────┼────────────────────┼─────────────────────────────────────┤
  │ GET      │ /api/v1/users      │  获取用户列表 (分页/过滤/排序)      │
  │ GET      │ /api/v1/users/:id  │  获取单个用户详情                   │
  │ POST     │ /api/v1/users      │  创建新用户                         │
  │ PUT      │ /api/v1/users/:id  │  全量更新用户                       │
  │ PATCH    │ /api/v1/users/:id  │  部分更新用户                       │
  │ DELETE   │ /api/v1/users/:id  │  删除用户 (软删除)                  │
  └──────────┴────────────────────┴─────────────────────────────────────┘

  ┌─ 响应格式 ──────────────────────────────────────────────────────────┐
  │                                                                     │
  │  成功:  { "code": 0, "data": {...}, "msg": "ok" }                   │
  │  失败:  { "code": 40001, "data": null, "msg": "参数错误" }          │
  │  分页:  { "code": 0, "data": [...], "total": 100, "page": 1 }       │
  │                                                                     │
  └─────────────────────────────────────────────────────────────────────┘

  ┌─ 状态码约定 ────────────────────────────────────────────────────────┐
  │                                                                     │
  │  200 OK          请求成功     │  400 Bad Request   参数错误         │
  │  201 Created     创建成功     │  401 Unauthorized  未认证           │
  │  204 No Content  删除成功     │  403 Forbidden     无权限           │
  │  301 Redirect    永久重定向   │  404 Not Found     资源不存在       │
  │  304 Not Modified 未修改      │  429 Too Many      请求过频         │
  │                               │  500 Server Error  服务器错误       │
  └─────────────────────────────────────────────────────────────────────┘
