║                        OAuth 2.0 授 权 码 流 程                       ║
╚═══════════════════════════════════════════════════════════════════════╝

  ┌────────┐                                          ┌──────────────┐
  │  用户  │                                          │  授权服务器  │
  │ (User) │                                          │ (Auth Server)│
  └───┬────┘                                          └──────┬───────┘
      │          ┌────────────────┐                          │
      │    1.    │   客户端应用   │                          │
      ├────────▸ │   (Client)     │                          │
      │  访问    └───────┬────────┘                          │
      │                  │                                   │
      │            2. 重定向到授权页                         │
      │◂─────────────────┤                                   │
      │                  │                                   │
      │     3. 用户登录并授权                                │
      ├──────────────────┼──────────────────────────────────▸│
      │                  │                                   │
      │     4. 返回授权码 (code)                             │
      │◂─────────────────┼───────────────────────────────────┤
      │                  │                                   │
      │            5. 用授权码换 Token                       │
      │                  ├──────────────────────────────────▸│
      │                  │                                   │
      │            6. 返回 Access Token + Refresh Token      │
      │                  │◂──────────────────────────────────┤
      │                  │                                   │
      │     7. 携带 Token 访问资源                           │
      │◂─────────────────┤                                   │
      │                  │                                   │
      ▼                  ▼                                   ▼
  ┌────────┐     ┌───────────────┐                   ┌──────────────┐
  │ 受保护 │     │ 资源服务器    │                   │  Token 验证  │
  │  资源  │◂───▸│ (Resource API)│──────────────────▸│  JWT / 内省  │
  └────────┘     └───────────────┘                   └──────────────┘
