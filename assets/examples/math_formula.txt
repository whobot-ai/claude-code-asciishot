  ┌─ 1. 前向传播 ───────────────────────────────────────────────────────────┐
  │                                                                         │
  │   输入 x ──▸ 线性变换 z = Wx + b ──▸ 激活 a = σ(z) ──▸ 输出 ŷ           │
  │                                                                         │
  └─────────────────────────────────────────────────────────────────────────┘

  ┌─ 2. 损失函数 ───────────────────────────────────────────────────────────┐
  │                                                                         │
  │              L(y, ŷ) = -[y·log(ŷ) + (1-y)·log(1-ŷ)]                     │
  │                                                                         │
  │                    (Binary Cross-Entropy)                               │
  │                                                                         │
  └─────────────────────────────────────────────────────────────────────────┘

  ┌─ 3. 反向传播 (链式法则) ────────────────────────────────────────────────┐
  │                                                                         │
  │    ∂L     ∂L    ∂ŷ    ∂z                                                │
  │   ─── = ───── · ── · ───                                                │
  │    ∂W    ∂ŷ     ∂z    ∂W                                                │
  │                                                                         │
  │        = (ŷ - y) · σ'(z) · x                                            │
  │                                                                         │
  └─────────────────────────────────────────────────────────────────────────┘
