            │               ▲                      ▲           ▲

  develop  ●───●───●───●───●───●───●───●───●──────●───●───●───● (integration)

  ── 分支说明 ─────────────────────────────────────────────────────────
  │ main      │ 生产环境，只接受 merge，永远可部署                    │
  │ develop   │ 集成分支，日常开发汇总                                │
  │ feature/* │ 功能分支，从 develop 拉出，完成后合回                 │
  │ hotfix/*  │ 紧急修复，从 main 拉出，修完同时合回 main 和 develop  │
  └───────────┴───────────────────────────────────────────────────────┘

  工作流: feature → develop → (测试通过) → main → 自动部署
